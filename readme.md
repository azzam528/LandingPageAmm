# 🎵 SPECTRA FEST 2026

Landing page website pemesanan tiket konser dengan tema **Neo Brutalism**, dibuat menggunakan **HTML, CSS, dan JavaScript**. Website ini menampilkan informasi festival musik, lineup artis, jadwal tampil, galeri konser, lokasi venue, FAQ, serta sistem pemesanan tiket.

## 📌 Tentang Project

SPECTRA FEST 2026 merupakan konsep website festival musik Indonesia yang menggabungkan desain modern dengan gaya **Neo Brutalism** untuk menciptakan pengalaman visual yang kuat dan menarik perhatian pengguna.

Website ini dirancang agar pengunjung dapat:

- Melihat informasi festival
- Mengecek lineup artis
- Melihat jadwal konser
- Menjelajahi galeri konser
- Memilih paket tiket
- Mengetahui lokasi venue
- Membaca FAQ acara

---

## 🎨 Mengapa Menggunakan Tema Neo Brutalism?

Tema **Neo Brutalism** dipilih karena memiliki karakter:

✅ Warna kontras tinggi  
✅ Border tebal dan bayangan keras  
✅ Tipografi besar dan berani  
✅ Tampilan unik dan energik  
✅ Memberikan kesan ekspresif dan modern  

Karakter tersebut cocok dengan **website pemesanan tiket konser**, karena konser identik dengan:

- Energi tinggi
- Lampu panggung
- Keramaian penonton
- Musik yang dinamis
- Pengalaman visual yang kuat

Desain Neo Brutalism membantu menciptakan suasana festival sejak pengguna pertama kali membuka website sehingga pengalaman pengguna terasa lebih hidup dan berbeda dibanding desain konvensional.

## ✨ Fitur

### Homepage
Menampilkan identitas festival serta informasi utama.

### Experience Section
Informasi pengalaman yang didapat pengunjung selama festival.

### Lineup Artist
Daftar artis yang tampil.

### Gallery Concert
Galeri suasana konser.

### Schedule / Rundown
Jadwal penampilan tiap hari.

### Ticket Section
Pilihan paket tiket:

- Regular
- 3 Day Pass
- VIP

### Venue
Informasi lokasi acara.

### FAQ
Pertanyaan umum pengunjung.

### Responsive Design
Website dapat menyesuaikan tampilan pada desktop maupun perangkat mobile.

### Hamburger Menu
Navigasi mobile menggunakan menu hamburger.
